# docs/guides 模块说明

> 最后更新: 2026-04-23
> 状态: 当前有效模块文档

## 1. 模块职责

- 记录仓库级创作与使用指南
- 把 CLI 工作流翻译成可执行的作者 / AI 操作步骤
- 为外部 AI 提供 canonical 流程、fallback 和停止条件

## 2. Owns

- `docs/guides/*.md`

## 3. 不变量

- 指南必须和 CLI 真实能力一致
- 流程说明必须包含完整闭环，而不是只给第一步
- 风格/类型建议不得与 `project.yaml` / `review` 口径冲突

## 4. 文档同步触发条件

- 工作流变化
- 命令矩阵变化
- 外部 AI 执行口径变化
