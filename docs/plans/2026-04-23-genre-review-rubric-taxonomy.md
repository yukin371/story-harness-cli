# 类型化评审标准表设计

> 日期: 2026-04-23
> 状态: 已落地

## 背景

现有 `review chapter` 只提供通用章节回顾分数，尚未处理“不同小说类型的评审重点不同”这一问题。

用户进一步提出：

1. 需要联网搜索常见小说评判标准
2. 需要把更常见的网络小说类型纳入，如轻小说、西幻、玄幻

## 设计结论

### 1. 不采用“所有标签平级”的分类法

不同标签的性质并不相同：

- `悬疑 / 言情 / 科幻 / 历史 / 纯文学` 更像主叙事类型
- `西幻 / 玄幻` 更像幻想大类下的重要子类型
- `轻小说` 更像风格 / 市场 / 受众层标签，不宜直接与悬疑、言情同层

### 2. 采用三层评审结构

1. 通用底盘：所有小说都评
2. 主类型门槛：不同主类型先过硬条件
3. 子类型 / 风格加权：根据细分流派调节权重

## 为什么要补 `轻小说 / 西幻 / 玄幻`

### 轻小说

有必要纳入，但建议放在“风格 / 市场标签”层：

- 学术资料普遍将 light novel 视为面向青少年、深受日本亚文化影响的出版标签或风格流
- 它会显著影响叙事方式、人物出场效率、语气、信息密度、章节钩子
- 但它本身不足以替代“故事主类型”

### 西幻

有必要纳入，建议放在“幻想子类型”层：

- 在中文网络文学和出版讨论里，西方奇幻通常被视为奇幻 / 玄幻谱系中的明确子类
- 其评审重点与一般奇幻不同，更强调世界规则、种族文明、史诗尺度、地图感与 lore 负载

### 玄幻

有必要纳入，建议放在“幻想子类型”层：

- 中国网络文学研究普遍承认玄幻是重要本土幻想类型
- 其类型特征常涉及升级驱动、体系感、奇观化、爽点、混合传统神话与外来幻想资源
- 与一般西幻、仙侠、武侠相比，评审权重确实不同
